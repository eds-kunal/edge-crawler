Usage Instructions

  1. Click "Start Crawling" on any of the examples above
  2. Monitor Progress as the crawler discovers pages
  3. Review Results showing internal and external links
  4. Export Data using the export buttons for further analysis

What the Crawler Does

  • 🔍 Discovers URLs from HTML content and sitemaps
  • 📊 Categorizes Links as internal or external
  • 🛡️ Handles CORS using proxy services when needed
  • 📥 Exports Results in multiple formats
  • ⚡ Rate Limits requests to be respectful

Expected Results

Each crawler will find:

  • Internal Pages: All pages within the same domain
  • External Links: Links to other websites and resources
  • Errors: Any pages that couldn't be accessed

💡 Tip: Use the export functionality to save results for further analysis in Excel, SEO tools, or other applications.

⚠️ Note: Some websites may block crawling due to CORS policies. The crawler will attempt to use proxy services but may still encounter restrictions on certain domains.
